## Overview

Sensei Advanced is a system optimization and monitoring application that provides hardware performance monitoring, disk cleanup, thermal tracking, battery health diagnostics, and system benchmarking in a beautifully designed native interface. It gives users comprehensive visibility into system health and performance with tools to maintain optimal operation, combining the monitoring depth of technical tools with the visual polish that makes the information accessible.

Hardware monitoring displays CPU, GPU, memory, disk, and network utilization with real-time graphs and historical data that reveal performance patterns. The monitoring covers per-core CPU usage, GPU utilization and temperature, memory pressure and swap usage, and disk I/O rates. The dashboard provides a complete performance overview at a glance.

Disk cleanup identifies and removes system caches, application leftovers, log files, and other expendable data that accumulates over time and consumes storage space. The cleanup presents found items by category with size information, enabling informed decisions about what to remove. Safe cleanup targets only expendable data that does not affect system or application function.

Thermal and battery monitoring tracks temperature sensor readings, fan speeds, and battery health metrics including cycle count, capacity degradation, and charge status. The thermal monitoring identifies overheating patterns, and battery health tracking builds awareness of battery aging that informs replacement decisions.

For users who want comprehensive system health monitoring with optimization tools in an interface that is informative without being overwhelming, Sensei Advanced is the system tool that makes hardware awareness accessible and actionable.

## Performance

Sensei balances performance with system resource usage intelligently. Operations are handled in the background, keeping your system responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series chips, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
